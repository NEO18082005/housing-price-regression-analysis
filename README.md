# House Price Prediction

This project was developed as a Week 1 internship assignment. The goal was to build a machine learning model to estimate property values based on real-world housing data, moving beyond guesswork to data-driven insights.

## 🚀 Project Overview
- **Objective:** Build a regression model to predict house prices.
- **Tools:** Python (Pandas, Scikit-learn, Matplotlib, Seaborn).
- **Models:** Linear Regression, Random Forest Regressor.

## 🛠️ Tasks Completed
1. **Data Exploration:** Loaded and explored dataset dimensions and features.
2. **Data Cleaning:** Handled missing values, removed duplicates, and performed one-hot encoding on categorical data.
3. **Model Building:** Trained and evaluated Linear Regression and Random Forest models using MAE, RMSE, and R² Score.
4. **Visualization:** Generated distributions, correlation heatmaps, and scatter plots to identify key pricing drivers.
